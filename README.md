# Usage
### Cocoapods
WSModel is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WSModel'
```

### Source code
把Classes文件夹下的所有文件拖入工程中。引入头文件`#import NSObject+WSJsonToModel.h`
